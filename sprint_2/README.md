# Informações

Nessa sprint aprendi sobre SQL, aprendi como rodar as queries, os principais comandos para selecionar tabelas e conseguir fazer consultas mais precisa de informações, como criar tabelas, inserir dados nela, criar visualizações, além de conceitos de dados, como ETL, Data Lake, Warehouse, modelo relacional x dimensional, entre outros.
